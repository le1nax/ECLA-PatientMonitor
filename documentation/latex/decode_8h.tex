Decompresses the input stream to the output stream. 

The values {\ttfamily \texorpdfstring{$\ast$}{*}available\+\_\+in} and {\ttfamily \texorpdfstring{$\ast$}{*}available\+\_\+out} must specify the number of bytes addressable at {\ttfamily \texorpdfstring{$\ast$}{*}next\+\_\+in} and {\ttfamily \texorpdfstring{$\ast$}{*}next\+\_\+out} respectively. After each call, {\ttfamily \texorpdfstring{$\ast$}{*}available\+\_\+in} will be decremented by the amount of input bytes consumed, and the {\ttfamily \texorpdfstring{$\ast$}{*}next\+\_\+in} pointer will be incremented by that amount. {\ttfamily total\+\_\+out}, if it is not a null-\/pointer, will be set to the number of bytes decompressed since the last {\ttfamily state} initialization.

\begin{DoxyNote}{Note}
Input is never overconsumed, so {\ttfamily next\+\_\+in} and {\ttfamily available\+\_\+in} could be passed to the next consumer after decoding is complete.

Acquires pointer to internal output buffer. 

This method is used to make language bindings easier and more efficient\+:
\begin{DoxyEnumerate}
\item push data to \doxylink{decode_8h_accb83dd5dab11ad1260c6f961a2340b3}{Brotli\+Decoder\+Decompress\+Stream}, until \doxylink{decode_8h_a87fd23715d225d1518eb7cd2a5cb374eafa10e01bd2f25a3d8d4b22db6fe66ab1}{BROTLI\+\_\+\+DECODER\+\_\+\+RESULT\+\_\+\+NEEDS\+\_\+\+MORE\+\_\+\+OUTPUT} is reported
\item use \doxylink{decode_8h_a263c9ba2f3acb679fae4f8dc8bfdfb99}{Brotli\+Decoder\+Take\+Output} to peek bytes and copy to language-\/specific entity
\end{DoxyEnumerate}

Also this could be useful if there is an output stream that is able to consume all the provided data (e.\+g. when data is saved to file system).

\begin{DoxyAttention}{Attention}
After every call to \doxylink{decode_8h_a263c9ba2f3acb679fae4f8dc8bfdfb99}{Brotli\+Decoder\+Take\+Output} {\ttfamily \texorpdfstring{$\ast$}{*}size} bytes of output are considered consumed for all consecutive calls to the instance methods; returned pointer becomes invalidated as well.
\end{DoxyAttention}
\begin{DoxyNote}{Note}
Decoder output is not guaranteed to be contiguous. This means that after the size-\/unrestricted call to \doxylink{decode_8h_a263c9ba2f3acb679fae4f8dc8bfdfb99}{Brotli\+Decoder\+Take\+Output}, immediate next call to \doxylink{decode_8h_a263c9ba2f3acb679fae4f8dc8bfdfb99}{Brotli\+Decoder\+Take\+Output} may return more data.
\end{DoxyNote}
